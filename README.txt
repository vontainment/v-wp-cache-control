
V-Cache Control Plugin
======================
Adds custom cache control headers to WordPress pages based on user roles and individual page settings.

Installation
------------
1. Upload the plugin files to the `/wp-content/plugins/v-cache-control` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.

Usage
-----
- Set global cache control headers for different user roles.
- Override cache control settings for individual posts, pages, and products using the provided metabox.
